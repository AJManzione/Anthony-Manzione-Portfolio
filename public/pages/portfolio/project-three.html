About Pokedex Tracker



Pokedex Tracker is a full stack applictaion designed for pokemon fans and gamers to keep track of their overall pokemon collections across all the main series games. The interactive user friendly interface allows a user to create an account and have their information stored into a mongoDB database using Bcrypt to hash their sensitive information. When a user signes up or logs in they are sent to the dashboard where they can customize their account by choosing a sprite and begin their journey adding pokemon to their collection. The app was created using React Native.



My Role

My role in this project was to make a responsive user friendly front end that feels fun and easy to use. I was responsible for compiling the data for the app which was done using a web scraper that I created specifically for this project. I created several JSON files from that scraper. I later created fetch requests to those JSON files and returned the data within each React component. Along with the fetch requests I was tasked with bringing together every visual aspect and styling aspect that you see on the website. Using bootstrap and useState and useEffect hooks. I created a dynamic experience that speaks to the database.
