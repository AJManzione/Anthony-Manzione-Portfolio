Team Profile Generator

- About this project -

The team profile generator is a CLI application that uses the inquirer npm package to prompt the user with questions in the node terminal. Based on these questions the user can build a team of employees and when done an html page will be generated with the team they selected. The user also has the option to select a color theme for their page

README.md Generator

- About this project -

The README.md Generator is a CLI aplication that utilizes the inquirer npm package to prompt the user with a series of questions in the node terminal. The user is asked to input information on a project that they are working on such as title, description, license type and so much more. Based on the answers given a README markdown file will be created so that the user has a great starting point for a README file in their next repository.
